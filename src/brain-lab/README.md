# Brain Lab - 大脑实验室

## 这是什么？

一个能看到AI"思考过程"的小游戏。

## 核心概念

### 1. 场景
```
🦊 → 想去 → 🏁
    但是 👿 挡住了
    跳上 🟩 踩 🔘 可以杀死 👿
```

### 2. 大脑 vs 小脑

| | 小脑 | 大脑 |
|--|------|------|
| 反应 | 看到敌人就躲 | 想象"跳上去踩按钮"的后果 |
| 结果 | 永远到不了终点 | 能到达终点 |

### 3. 你能看到什么？

**游戏区**（左）：
- 🦊 狐狸（蓝色圆圈）
- 👿 敌人（红色方块）
- 🟩 平台（绿色）
- 🔘 按钮（紫色）
- 🏁 终点（橙色）

**大脑区**（右）：
```
🧠 大脑正在思考...

选择「跳跃」，因为：
可以到达按钮平台触发机关清除敌人
（其他选项如右移预测会遇到敌人）

┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐
│ ⬅️ 左移  │ │ ➡️ 右移  │ │ ⬆️ 跳跃 │ │ ⏸️ 等待 │
│ 预测:-10 │ │ 预测:-5  │ │ 预测:+50│ │ 预测:0  │
│         │ │         │ │ ✨选中  │ │         │
└─────────┘ └─────────┘ └─────────┘ └─────────┘
```

## 怎么运行？

```bash
# 在项目根目录
npm run dev

# 或者用 vite 直接运行
cd configs
npx vite --config vite.config.ts
```

然后浏览器打开：
```
http://localhost:4000/pages/brain-lab.html
```

## 怎么玩？

1. **点"单步"**：看AI思考一步
2. **看右侧面板**：看AI想象了什么
3. **点"自动"**：让AI自己解决问题
4. **调滑块**：改变AI"看多远"（想象深度）

## 文件说明

| 文件 | 作用 |
|------|------|
| World.ts | 游戏世界，管理地图和规则 |
| WorldModel.ts | 世界模型，AI的"想象力" |
| Brain.ts | 大脑，做决策 |
| Renderer.ts | 画图，让你看到AI在想什么 |
| main.ts | 程序入口 |

## 关键代码（最简版）

```typescript
// Brain.ts - 大脑的核心
class Brain {
  think(当前状态) {
    // 想象每种可能
    const 想象结果 = [
      { 动作: "左移", 预测位置: (0,1), 预测奖励: -10 },
      { 动作: "右移", 预测位置: (2,1), 预测奖励: -5 },  // 有敌人
      { 动作: "跳跃", 预测位置: (3,0), 预测奖励: +50 }, // 能踩按钮！
      { 动作: "等待", 预测位置: (1,1), 预测奖励: 0 },
    ]
    
    // 选最好的
    return 想象结果中(奖励最高的)
  }
}
```
