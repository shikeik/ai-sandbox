# 奖励塑形是强化学习收敛的前提

## 场景
想让 AI 通过试错自主学习，而不是直接给正确答案。

## 核心经验
**仅靠"死了-1、活了+1"的稀疏奖励，AI 几乎学不会任何复杂任务。**

必须在每一步都给出密集的中间奖励（Reward Shaping），例如：
- 每安全存活一步：+1
- 吃到金币：+50
- 正确躲避坑：+10
- 掉坑/撞敌：-200

## 为什么有效
稀疏奖励下，AI 99% 的时间在死，且无法区分"运气"和"实力"。密集奖励让 AI 至少能学到"多活一步是好的"，建立起最基础的梯度信号。

## 组合技
奖励塑形通常需要和以下技术一起用：
1. **课程学习**：从简单地形开始，逐步解锁复杂规则
2. **经验回放**：存储 `(s, a, r, s')` 并随机抽取训练
3. **优先回放**：重点学习"意外事件"（高分/猝死）
4. **模仿学习启动**：先用少量监督数据给网络一个正确先验
