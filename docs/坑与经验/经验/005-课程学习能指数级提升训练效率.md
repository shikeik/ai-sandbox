# 课程学习能指数级提升训练效率

## 场景
AI 需要在复杂环境中做决策，直接训练最终难度往往无法收敛。

## 核心经验
**从最简单的情况开始，逐步增加难度**，比直接在最难环境上训练快得多。

## 示例课程设计
```
阶段 1：只有平地 → 学会"走"
阶段 2：偶尔出现单坑 → 学会"跳"
阶段 3：混合地形（坑+平地）
阶段 4：加入史莱姆、恶魔、金币
```

## 效果
没有课程学习时，AI 会在最难地形上浪费 99% 的尝试。有了课程学习：
- 早期快速建立基础能力
- 中期逐步泛化
- 后期处理复杂组合

## 迁移学习 bonus
课程学习通常让网络权重跨阶段保留（迁移学习），后一阶段的收敛速度会更快。
