# 监督学习在规则清晰、数据质量高时效率极高

## 场景
任务规则明确，可以写出"正确答案"（如地形跳跃、棋类走法、代码补全）。

## 核心经验
当满足以下条件时，监督学习可以在浏览器里秒级收敛：
1. **规则清晰**：决策边界明确，没有模糊地带
2. **数据质量高**：样本覆盖全面，标签 100% 正确
3. **网络规模匹配**：参数量与任务复杂度成正比（不要大材小用）
4. **分类目标简单**：类别数少（如 4 类动作）

## 量化参考
| 条件 | 收敛所需样本 |
|---|---|
| 规则生成数据 + 小型 MLP（90→32→4） | 6,000 条 |
| 相同任务，纯强化学习（无课程） | 可能永远学不会 |

## 应用建议
如果项目目标是"教学演示"或"快速验证"，优先用监督学习/模仿学习：
- 先用规则 AI 生成带标签的训练数据
- 让神经网络快速学会"大概正确的策略"
- 如果需要，再在这个先验基础上做强化学习微调
