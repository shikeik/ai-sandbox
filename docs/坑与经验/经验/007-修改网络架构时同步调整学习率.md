# 修改网络架构时同步调整学习率

## 场景
把神经网络隐藏层从 16 神经元改成 32 神经元，希望提升表达能力。

## 坑
如果只改 `hidden_dim` 而不改学习率，可能出现：
- 学习率太小 → 收敛极慢
- 学习率太大 → 梯度爆炸或震荡

## 经验公式
对于这个项目的小规模 MLP，可维持近似比例：
```
LR × HIDDEN_DIM ≈ 3.2
```

示例：
| hidden_dim | 建议 LR |
|---|---|
| 16 | 0.2 |
| 32 | 0.1 |
| 64 | 0.05 |

## 原理
参数量翻倍后，梯度累积的期望幅值也会变化。适当降低学习率可以保持更新的稳定性。

## 注意
这只是一个经验性起点，实际最优学习率仍需通过收敛速度和最终 loss 来验证。
