# Brain Lab 快速开始

## 一句话说明

**Brain Lab 是一个能看到AI"思考过程"的小游戏。**

## 运行步骤

### 1. 启动服务器
```bash
cd configs
npx vite --config vite.config.ts
```

### 2. 打开浏览器
```
http://localhost:4000/pages/brain-lab.html
```

## 你会看到什么

### 左侧：游戏世界
```
🦊 → 想去 → 🏁
    但是 👿 挡住了
    跳上 🟩 踩 🔘 可以杀死 👿
```

- 🦊 狐狸（蓝色）- 主角
- 👿 敌人（红色）- 挡路的
- 🟩 平台（绿色）- 可以站
- 🔘 按钮（紫色）- 踩了会触发机关
- 🏁 终点（橙色）- 目标

### 右侧：大脑思考
```
🧠 大脑正在思考...

选择「跳跃」，因为：
可以到达按钮平台触发机关清除敌人
（其他选项如右移预测会遇到敌人）

┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐
│ ⬅️ 左移  │ │ ➡️ 右移  │ │ ⬆️ 跳跃 │ │ ⏸️ 等待 │
│ 预测:-10 │ │ 预测:-5  │ │ 预测:+50│ │ 预测:0  │
│         │ │         │ │ ✨选中  │ │         │
└─────────┘ └─────────┘ └─────────┘ └─────────┘
```

## 怎么玩

1. **点"单步"** - 看AI思考一步
2. **看右侧** - 看AI想象了什么
3. **点"自动"** - 让AI自己解决问题

## AI的思考逻辑

```
每一步，AI都会：

1. 想象："如果我左移，会发生什么？"
   → 预测位置 → 预测奖励

2. 想象："如果我右移，会发生什么？"
   → 预测位置 → 预测奖励

3. 想象："如果我跳跃，会发生什么？"
   → 预测位置 → 预测奖励（最高！因为能到按钮）

4. 想象："如果我等待，会发生什么？"
   → 预测位置 → 预测奖励

5. 选择奖励最高的动作
```

## 关键：为什么需要大脑？

**普通AI（小脑）**：
- 看到敌人 → 只会躲 → 永远到不了终点

**有大脑的AI**：
- 想象"跳上去踩按钮"的后果
- 发现可以杀死敌人
- 选择跳跃 → 到达终点

**这就是大脑的价值：能在"脑子里"模拟未来，而不是只能反应现在。**

## 调滑块

**想象深度滑块**（1-5）：
- 1：只看下一步
- 3：看后面3步（推荐）
- 5：看后面5步

深度越大，AI考虑得越长远，但计算也更慢。

## 如果出问题了

1. 狐狸跳不上去？
   - 刷新页面重置
   
2. 右侧大脑面板不显示？
   - 按F12看控制台有没有报错
   
3. AI不杀敌人？
   - 点"重置"再试一次
