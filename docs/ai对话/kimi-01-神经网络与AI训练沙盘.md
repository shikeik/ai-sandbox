## 一、聊了哪些内容

### 1. AI技术流派讨论
- 用户最初认为AI训练都是基于"参数节点网络"+梯度下降
- 我补充了其他流派：进化计算、强化学习、符号主义、传统统计学习
- 用户确认关注"通过训练成长"的AI，最终聚焦：**连接主义(神经网络)**为主，辅以进化算法

### 2. 原始神经元学习模型
- 剥离所有工程复杂性，回到最本质：输入→处理→输出，通过生存反馈调整连接
- 核心洞察：**不需要人为记分，生存vs死亡就是最自然的信号**
- 三要素框架：目标、信息、能力

### 3. 沙盘游戏设计
经过多次迭代，最终确定：

**游戏世界**：
- 32格长度的横版世界
- 地面(y0)有平地(可站)和坑(掉落死亡)
- 玩家(y1站立, y2跳跃中)
- 终点在x30

**操作**：
- 只有「右移」(x+1)和「跳跃」(x+2，抛物线)
- 移动中踩坑→死亡
- 跳跃落点坑上→死亡

**相机**：
- 玩家位于屏幕左侧30%位置
- 玩家走到屏幕中间才触发相机跟随
- 世界格子用淡淡竖线标记（每100px一格）

**地形生成**：
- 禁止连续坑洞（玩家无法跳过）
- 65%概率平地(1-2格)，35%概率坑(1格)

## 五、技术约束

### 环境限制（Termux）
- ❌ 不能安装 PyTorch/TensorFlow（无ARM64预编译包）
- ❌ npm install 全局包要在 `~` 目录（/storage/emulated/0/ 有权限限制）
- ✅ 使用纯JavaScript实现神经网络（不依赖外部库）

### 运行方式
```bash
cd ai-sandbox

# 方式1：全局vite（推荐）
cd ~ && npm install -g vite
cd /storage/emulated/0/AppProjects/html/ai-sandbox
vite --host

# 方式2：npm脚本（如果全局安装了vite）
npm run dev

# 访问 http://localhost:4000
```
