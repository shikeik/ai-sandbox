# AI 交接文档目录

存放各阶段开发交接文档。

## 文档命名规范

### 最终命名
```
kimi-XX-中文标题.md
```

- `kimi-` 前缀固定
- `XX` 为两位数字序号（01, 02...）
- 标题使用中文，简明概括主题
- 示例：`kimi-10-地形实验室课程学习.md`

### 进行中命名
当对话产生阶段性总结但**尚未结束时**，应立即创建进行中文档：

```
kimi-XX-进行中.md
```

待对话完全结束后，再重命名为最终标题：

```
kimi-XX-最终中文标题.md
```

**目的**：避免对话还没结束就猜测最终标题，导致文档名与内容不匹配。
