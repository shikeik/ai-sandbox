## 状态流转

```
页面加载
    ↓
init() 初始化
    ↓
STATUS = READY, _inputLocked = true
    ↓
显示开始遮罩
    ↓
玩家点击"开始游戏"
    ↓
startGame() 被调用
    ↓
STATUS = IDLE, _inputLocked = false
    ↓
隐藏遮罩，启动计时器
    ↓
游戏正式开始
```
