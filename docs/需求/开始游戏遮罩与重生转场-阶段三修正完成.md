# 阶段三修正完成：重生后直接开始游戏

## 修正说明

之前的实现有错误：重生后显示了开始遮罩，需要玩家再次点击才能开始。根据需求，重生后应该直接开始新一局。

## 问题修复

### 1. 修改 `src/main.js` 中的 `onTransitionEnd`

**修复前：**
```javascript
game.onTransitionEnd = () => {
  if (!isAIMode) {
    // 玩家模式：显示开始遮罩，等待玩家点击
    showStartOverlay()
    // 重置计时器
    game.startTime = null
    stopTimerUpdate()
    updateGameInfo()
  } else {
    // AI 模式：自动开始新世代
    game.startGame()
    startAI()
  }
}
```

**修复后：**
```javascript
game.onTransitionEnd = () => {
  if (!isAIMode) {
    // 玩家模式：重生后直接开始新一局（不显示遮罩）
    game.startGame()      // 启动计时器，解锁输入
    startTimerUpdate()    // 启动 UI 更新
  } else {
    // AI 模式：自动开始
    game.startGame()
    startAI()
  }
}
```

### 2. 修改 `src/main.js` 中的 `init()`

在初始化末尾添加首次加载显示遮罩：

```javascript
function init() {
  // ... 所有初始化代码 ...
  
  // 首次加载显示开始遮罩
  showStartOverlay()
  
  console.log('🎮 AI 训练沙盘已初始化，等待开始...')
}
```

### 3. 修改 `src/game/JumpGame.js` 中的 `startGame()`

支持从 `TRANSITIONING` 状态启动：

```javascript
/**
 * 开始游戏（从 READY 或 TRANSITIONING 状态进入 IDLE）
 */
startGame() {
  if (this.player.status === STATUS.READY || this.player.status === STATUS.TRANSITIONING) {
    this.player.status = STATUS.IDLE
    this._inputLocked = false
    this.startTimer()  // 重置并启动计时器
  }
}
```

## 正确的行为

| 场景 | 行为 |
|------|------|
| **首次进入页面** | 显示开始遮罩 → 点击后第一局开始 |
| **玩家死亡/胜利** | 转场（渐暗渐亮）→ 直接开始新一局 |
| **AI 模式** | 转场 → 自动继续下一局 |

## 状态流转图

**首次加载：**
```
页面加载 → init() → 显示遮罩 → 点击开始 → startGame() → 游戏进行
                                         ↓
                                    启动计时器
```

**重生流程：**
```
死亡/胜利 → 等待800ms → 渐暗400ms → 重生(暗屏) → 渐亮400ms → onTransitionEnd()
                                                                    ↓
                                                              startGame()
                                                                    ↓
                                                              启动计时器，游戏继续
```

## 测试验证

1. **首次加载**：刷新页面 → 显示遮罩 → TIME 显示 `--:--` → 点击开始 → TIME 开始计时
2. **死亡重生**：死亡 → 转场 → 直接开始新一局 → TIME 重置并立即计时
3. **胜利重生**：胜利 → 转场 → 直接开始新一局 → TIME 重置并立即计时
4. **AI 模式**：自动运行，转场后自动继续
